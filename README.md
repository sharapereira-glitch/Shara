# # 🃏 Meu Primeiro Cartão com HTML e CSS

Este é o resultado do meu desafio prático de desenvolvimento web, onde explorei o alinhamento e o comportamento do **Box Model** no CSS.

## 🚀 O que eu aprendi

Neste projeto, pratiquei como estruturar um elemento em HTML e estilizá-lo usando CSS, focando em:
* **`width` e `height`:** Para controlar o tamanho exato do cartão.
* **`padding`:** Para dar espaço interno e não deixar o texto colado nas bordas.
* **Integração:** Como conectar o arquivo `index.html` ao `style.css`.

## 🧪 Os Testes do Desafio

Durante o desafio, alterei as seguintes propriedades para ver o comportamento do layout:
* Testei o `width` com **260px** e **200px**.
* Experimentei o `padding` com **15px 25px** para dar mais respiro nas laterais.

> [!TIP]
> O resultado que eu mais gostei usou a combinação de `width: ______` e `padding: ______` porque achou que o visual ficou mais harmônico!
